A keyboard is a typewriter-style device, which uses an arrangement of buttons or keys, to act as mechanical levers or electronic switches. Following the decline of punch cards and paper tape, interaction via teleprinter-style keyboards became the main input device for computers.

The Cylon adaptor for keyboard input.

For more info about the Joystick platform click here.

How To Install

Installing cylon.js with a keyboard support is pretty easy.

npm install cylon-keyboard

How To Use

var Cylon = require('cylon');

Cylon.robot({
  connection: { name: 'keyboard', adaptor: 'keyboard' },
  device: {name: 'keyboard', driver: 'keyboard'},

  work: function(my) {
    my.keyboard.on('a', function(key) { 
      console.log("A PRESSED!");
    });
  }
}).start();

How to Connect

Plug the keyboard to your computer as usual.
