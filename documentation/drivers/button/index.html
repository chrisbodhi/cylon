How To Use

This example detects when a button has been pushed. 

var Cylon = require('cylon');

Cylon.robot({
  connection: {name: 'arduino', adaptor: 'firmata', port: '/dev/ttyACM0'},

  device: {name: 'button', driver: 'button', pin: 2},

  work: function(my) {
    my.button.on('push', function() {
      Logger.info("Button pushed!");
    });
  }
}).start();

Commands

isPressed()

Returns true or false matching state of the button.

Returns

true or false

Events

update

This event will contain one of the following events. You can also subscribe to one of the specific events, if that is the information you are interested in.

push

This event gets triggered when the button changes state from released to pushed.

release

This event gets triggered when the button changes state from pushed to released.

start

Sent when the device has been started and is ready to use.
