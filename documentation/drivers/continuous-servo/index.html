How To Use

This example moves a servo from angle 45 to 90 to 135. 

var Cylon = require('cylon');

Cylon.robot({
  connection: {name: 'arduino', adaptor: 'firmata', port: '/dev/ttyACM0'},

  device: {name: 'servo', driver: 'continuous-servo', pin: 3},

  work: function(my) {
    var angle = 45 ;
    my.servo.angle(angle);
    every((1).second(), function() {
      angle = angle + 45 ;
      if (angle > 135) {
        angle = 45
      }
      my.servo.angle(angle);
    });
  }
}).start();

Commands

clockwise()

Turns the servo to go clockwise, if the driver is continuous.

Returns

true or nil

counterClockwise()

Turns the servo to go counter clockwise, if the driver is continuous.

Returns

true or nil

stop()

Stops the driver.

Returns

nil

Events

start

Sent when the device has been started and is ready to use.
